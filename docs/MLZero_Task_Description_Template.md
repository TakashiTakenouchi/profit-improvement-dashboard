# MLZero タスク説明テンプレート

## 推奨タスク説明（依存関係エラーを防ぐ）

### 基本テンプレート

```
添付ExcelのDailyForecastDataシートを見て90日間の予測を作成してください。

重要: 
- Bashスクリプトの最初に「pip install autogluon.timeseries pandas openpyxl」を必ず追加してください
- 依存関係のチェックではなく、インストールを実行してください
- エラーで終了せず、インストールしてからコードを実行してください
```

### 詳細テンプレート

```
タスク: 時系列予測
データ: 添付ExcelファイルのDailyForecastDataシート
予測期間: 90日間
使用ライブラリ: autogluon.timeseries

実行要件:
1. Bashスクリプトの最初に以下のコマンドを追加してください:
   pip install autogluon.timeseries pandas openpyxl

2. 依存関係をインストールしてからPythonコードを実行してください

3. エラーで終了せず、インストールを実行してください

出力形式: CSVファイル
```

## よくある問題と対策

### 問題1: 依存関係がインストールされない

**原因**: MLZeroが生成したBashスクリプトが依存関係をチェックするだけで、インストールしない

**対策**: タスク説明で依存関係のインストールを明示的に指示

### 問題2: 仮想環境で依存関係が見つからない

**原因**: MLZeroが仮想環境を作成してコードを実行する場合、システム環境の依存関係が使用できない

**対策**: Bashスクリプトで依存関係をインストールする

## タスク説明のベストプラクティス

1. **依存関係のインストールを明示**
   - 「pip install」コマンドを具体的に指定

2. **実行順序を明確に**
   - 依存関係のインストール → コード実行

3. **エラーハンドリングを指示**
   - エラーで終了せず、インストールを実行

4. **必要なライブラリを列挙**
   - autogluon.timeseries
   - pandas
   - openpyxl
